# 💬Projet Tchat
Projet de tchat en temps réel fait avec Socket.io.

Le but est d'utiliser les WebSockets et la communication temps réel entre plusieurs utilisateurs.

# Tech utilisées

- React + Vite
- Tailwind CSS
- Node.js + Express
- Socket.io

Une base de données serait ajoutée ultérieurement

# Ce que l’application peut faire

- envoyer des messages en temps réel
- choisir un pseudo quand on arrive sur la première page
- afficher ses messages à droite et ceux des autres à gauche
- scroll automatique vers les nouveaux messages
- envoyer un message avec la touche Entrée

# Installation
## Backend

cd back
